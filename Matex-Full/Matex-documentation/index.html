Installing Template.

  1. After unzip the download pack, you'll found a Template Folder with all the files.
  2. You can view this Template in any browser, you can display or edit the Template without an internet connection.(May not wotrk fonts and google map).
  3. The only section that will not work is the Contact Section which contains PHP code and requires a server environment for send messages.
  4. Now go to your cpanel or open your FTP Client (like Filezilla) and upload the content of the Template on your server root.
  5. Once the files are done uploading go to www.yourdomainname.com/index.html

Contact Form

This is a PHP Script for sending messages to your email, you should replace support@pixelcoder.net to your email to start receive messages.

	                    $to = "support@pixelcoder.net"; // replace this mail with yours
	                    $from = $_SERVER['PHP_SELF']." ".$_POST["email"];
	                    $email = $_POST["email"];
	                    $fname = $_POST["fname"];
	                    $lname = $_POST["lname"];
	                    $headers = "From: $from";
	                    $phone = $_POST["phone"];
	                    $message = $_POST["message"];

	                    $body = "User Message \n";
	                    $body .= " \n\n\t Name: ".$fname." ".$lname;
	                    $body .= " \n\n\t Email: ".$email;
	                    $body .= " \n\n\t Phone: ".$phone;
	                    $body .= " \n\n\t Message: ".$message;

	                    if ( mail( $to, $subject, $body, $headers ) ) {
	                        echo '';
	                    } else {
	                        echo '';
	                    }

Google Map Settings

To set your location, go to template root directory, open js folder, find map-init.js edit it with text editor (like Notepad++). Go to line number 06 and change the latitude and longitude. First one is latitude and second one is longitude. To find your locations latitude and longitude, click here
